# VIJJU COOKING

A modern cooking website created for sharing recipes, food inspiration, and cooking videos. Features a cinematic dark theme, smooth animations, responsive design, recipe showcase cards, and direct links to the Vijju Junnu Official YouTube channel.

## Features

* Modern Dark & Gold Design
* Responsive Layout
* Animated Hero Section
* Recipe Showcase
* YouTube Integration
* Instagram DM Button
* Smooth Scrolling Effects
